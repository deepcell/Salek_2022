# Generate ROC Curve

This repository contains the script used to ROC curves

## Contents

- plot_roc.py - Script used to generate ROC curves
- demo_dataset - Small demo dataset containing labels and predictions
- ROC_Curve.png - Output ROC curve on demo dataset
- requirements.txt

## Steps to generate ROC curve

- Install the dependencies from requirements.txt
- Run plot_roc.py script as follows

```sh
python plot_roc.py --run_ids_file demo_dataset/runs.txt --predictions_folder demo_dataset/preds_parquets/ --labels_file demo_dataset/labels.parquet --pos_classes NSCLC --pos_labels NSCLC --neg_labels WBC
```
